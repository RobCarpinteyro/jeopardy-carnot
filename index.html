⚙️ Configurar Firebase

Para conectar los 150 dispositivos en tiempo real necesitas un proyecto gratuito de Firebase. Solo se hace una vez. Sigue estos pasos:

1. Ve a console.firebase.google.com 2. "Agregar proyecto" → nombre cualquiera → Continuar 3. Desactiva Analytics → Crear proyecto 4. En el menú izquierdo: Build → Realtime Database 5. "Crear base de datos" → elige región → modo prueba → Habilitar 6. Clic en el ícono </> (Web) en la pantalla principal 7. Registra la app → copia los valores de abajo

💡 La configuración se guarda en este dispositivo para la próxima sesión.
